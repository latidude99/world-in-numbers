## General Info

My second Python project.  
Presenting various data in charts and tables.  

Phase one: COVID-19  
Phase two: CIA Factbook  


## Technologies
- Python 3.8
- Django 3.1
- MySQL 5.7
- Bootstrap 4.5.2
- Bootstrap-select 1.13.14
- JQuery 3.5.1
- Chart.js 2.9.3
